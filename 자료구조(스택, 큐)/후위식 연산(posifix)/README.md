# 후위식 연산(postfix)
후위연산식이 주어지면 연산한 결과를 출력하는 프로그램을 작성하세요.<br>
만약 3*(5+2)-9 을 후위연산식으로 표현하면 352+*9- 로 표현되며 그 결과는 12입니다.<br>
<br>
■ 입력설명<br>
첫 줄에 후위연산식이 주어집니다. 연산식의 길이는 50을 넘지 않습니다.<br>
식은 1~9의 숫자와 +, -, *, / 연산잘뫈 이루어진다.<br>
<br>
■ 출력설명<br>
연산한 결과를 출력합니다.<br>
■ 입력예제 1<br>
352+*9-<br>
<br>
■ 출력예제 1<br>
12
