# 가운데 문자 출력
소문자로 된 단어(문자열)가 입력되면 그 단어의 가운데 문자를 출력하는 프로그램을 작성하세요. 단 단어의 길이가 짝수일 경우 가운데 2개의 문자를 출력합니다.<br>
<br>
■ 입력설명<br>
첫 줄에 문자열이 입력된다. 문자열의 길이는 100을 넘지 않습니다.<br>
<br>
■ 출력설명<br>
첫 줄에 가운데 문자를 출력합니다.<br>
<br>
■ 입력예제 1<br>
study<br>
<br>
■ 출력예제 1<br>
u<br>
<br>
■ 입력예제 2<br>
good<br>
<br>
■ 출력예제 2<br>
oo
